Enter the names of the sections you wish to exclude from the graphs, summary and total sum. Separate names by using a comma. Leave empty to not exclude any previously included sections.

There is limited support for wildcards:

  • asterix at the end of a section name: all sections with this prefix will be excluded.
  • asterix at the start of a section name: all sections with this suffix will be excluded.

Note that these wildcards can NOT combined and matching is case-insensitive
